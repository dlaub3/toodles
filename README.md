# Exploring go web development with Gin and Mgo


##  Setup

* `git clone https://github.com/dlaub3/toodles.git && cd toodles`
* `govendor sync`
* `docker-compose up -d`
* `fresh`

Visit

* [http://localhost:8081](Mongo-Express) `admin:password`
* [http://localhost:8080](Toodles)
